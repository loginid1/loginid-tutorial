Authentication

With FIDO2 users do not only register a username but also the used authenticator (i.e.: TouchID). After registering, a user can login to the website by using the registered authenticator, no password required.

Please note that device and authenticator are often user interchangeably in FIDO2!

This screen supports both features:

  1. Registering a username with an authenticator (device)
  2. Login with a username using a registered authenticator (device)

The Sign-In feature will figure out if the user is already known and will prompt for registration if needed!
