Add Authenticator

Adding an authenticator (device) is the task of enabling an additional device to use with an user account.

It requires three steps:

  • an authorization code has to be generated on the new device
  • the generated authorization code has to be authorized on the first device
  • register the authenticator on the new device

To execute this feature on a MacBook two different browsers can be used. Follow these steps:

  1. other browser - Request Adding Credential: provide your username and copy the generated authorization code.
  2. this browser - Add Authenticator: assuming this authenticator (device/browser) has been used to sign-up initially. If so, paste the code and click Authorize.
  3. other browser - Request Adding Credential: add the username, the authorized code and click Add Authenticator
