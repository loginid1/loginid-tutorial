Request Adding Authenticator

Adding an authenticator (device) is the task of enabling an additional device to use with an user account.

It requires three steps:

  • an authorization code has to be generated on the new device
  • the generated authorization code has to be granted on the first device
  • register the authenticator on the new device

To execute this feature on a MacBook two different browsers can be used. Follow these steps:

Open this page on the other browser!

  1. this browser - Request Adding Authenticator: sign-in and copy the generated authorization code.
  2. other browser - Add Authenticator: assuming that authenticator (device/browser) has been used to sign-up initially. If so, paste the code, click Authorize.
  3. this browser - Request Adding Authenticator: click Add Authenticator

Use the dialog below to generate an authorization code:

Once the authorization code was authorized, provide your username and the code again to complete the process.
